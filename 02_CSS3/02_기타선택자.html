1. 속성선택자

선택하고자 하는 요소 내에 작성돼있는 속성을 이용해서 선택하는 방법
선택자 뒤에 [ ]를 이용해서 속성과 속성값을 제시하면서 선택 

        선택자[속성=속성값]
        선택자[속성~=속성값] 
        선택자[속성|=속성값] 
        선택자[속성^=속성값] 
        선택자[속성$=속성값]
        선택자[속성*=속성값] {
            스타일속성:값;
        }

2. 자손선택자와 후손선택자

요소들이 중첩되어 작성 가능
자손: 바로 하위인 요소들을
후손: 하위요소들 전부 

     * 자손선택자: > (부모쪽으로 열림)
        a>b{
            스타일속성:값;
        } (a의 자식인 b를 선택)

     * 후손선택자: 공백
     a b {
        스타일속성:값;
     }   (a의 후손인 b)
