텍스트 관련 스타일(text~)

color: 텍스트의 색상을 지정

    선택자 {
        color:색상명 | 16진수 | rgb(x,y,z) | rgba(x,y,z,투명도) | hsl(x,y,z) | hsla(x,y,z,투명도)
    }

text-decoration: 텍스트에 줄을 긋거나 없앨 때 사용

    선택자 {
        text-decoration: none | underline | overline | line-through
    }

  • 텍스트 영역 위에 줄 긋기

text-transform: 영문 텍스트 대소문자 변환 시 사용

    선택자 {
        text-transform: uppercase | lowercase | capitalize
    }
  1. uppercase: 모든 영문자를 대문자로
  2. LowerCase: 모든 영문자를 소문자로
  3. capitalize: 영문자의 firsttext만 대문자로

text-shadow: 텍스트에 그림자 효과를 줄 때 사용

    선택자 {
        text-shadow: (가로거리)x (세로거리)y [번짐정도]
    }
